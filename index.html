Common Questions

How do I join the server?
Simply copy our IP (play.cartoonsmp.com) and add it to your Minecraft server list! We support Java Edition 1.20+.
Is the server free to play?
Yes! Cartoon SMP is completely free to play. We have an optional store for cosmetics to help support the server.
Can I play on Bedrock Edition?
Yes, we support cross-play! Bedrock players can join using the same IP with port 19132.
